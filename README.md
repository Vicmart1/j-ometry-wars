# j-ometry-wars
This is a java recreation of the classic Geometry Wars game as a final project for my AP Computer Science class.

### Usage
Run jometrywars.jar (Must have java installed)

### Controls
<ul>
<li>WASD Keys - Move</li>
<li>Arrow Keys - Fire</li>
<li>Space - Bomb</li>
</ul>
